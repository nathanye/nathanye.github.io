HTML 是什么？

HTML 是为 网页而生 的 标记语言 ，这门语言（或是符号）是你与网页对话的基础，但它 并不是 一门 编程语言 。

  • HTML 借助 标记语言 来描述整个网页框架
  • HTML 中各种元素能够构建出网页中的各个模块
  • HTML 中的各个元素，又是由各种不同的 标签 所代表体现出来
  • HTML 标签 有”heading””paragraph””table”等等
  • 浏览器 并不会显示标签，它会通过 标签 来显示网页中我们想要的内容

突破心障，是普通人活出自我的唯一途径。

在返回上海的高铁上翻完全书，合上书时开始沉浸在“心障”的概念中，在想自己在哪些方面给自己设有“心障”。

我们接受了长年的应试教育，很容易忘记学习的目的其实是改变自己，从而改善生活。我们长久以往的为了父母，为了分数，为了排名，为了……，其实学习这件事情根本只是关于我们自己一个人的事情罢了。可太多人从学校毕业以后就停止了成长、进步，也许他们通过中学时代的努力拼搏有着高校的光环，毕业以后拿着还不错的薪水，就这样安安稳稳的开始人生后面的路，然而我却没有他们的那些“优势”，也许我无法或是不允许自己那样碌碌无为、浑浑噩噩的混过这一生。从李笑来老师的《把时间当作朋友》中提到的“心智”和今天Tinyfool老师所说的“心障”能够感受到，普通人要想脱离不平凡唯有掌控“心智”、突破“心障”。

我们常常因为一些口口相传的惯性思维“侵蚀”我们的大脑，从而让我们自己潜移默化间为自己设置了许多“心障”。学了十几年的英文，在街上遇到问路的老外时却一时间只能从口中蹦出几个简单的单词，虽然也帮上了忙，却还是心存小遗憾。在地铁上看到能与外国人谈笑风生的陌生人时，内心总是心生羡慕之情，感叹自己英文太差。“如果当年好好背单词、认真学英语的话……”这样的句式是不是很熟悉？我们总是认为学习一门语言需要有老师或者有经验的人士来进行传授，却忘了我们牙牙学语时并没有老师的介入，只有“练习”、“重复”和“用”。在如今这样一个互联网高度发达的时代，获取学习语言需要的资料真的可以说是“触手可得”！

而就是身处这样的环境之中，我们也并非能够通过“用”英语来提高，因为我们太急于求成而又太过玻璃心了！刚得知一种新学习方法时往往被一股“迷之力量”打入一剂鸡血，信誓旦旦的开启了“学习之路”，可坚持了没几天发觉“啊！怎么这样难啊！完全看不懂、听不进去啊！这种方法真的有效吗……”我们或许太过玻璃心，遇到一点点高度的“障碍”就开始打退堂鼓。当然，也许是在注入“鸡血”时太过高估自己的自控力、意志力和学习能力，给自己订了一个有些挑战的目标，这样一旦没法达成就会给自己带来不小的伤害，从而抑制自己继续向前走。另一方面，浮躁的我们一味的追求“XX天学会XX”，在自己学习过程中如果超过了XX天就会给自己贴上“太笨”的标签，时常忘记上学时老师常常提到的“循序渐进”。其实学习是需要“积累”的，然而“积累”本身就需要花费很长时间，学习成果则是要在“积累”到一定时间段之后才会获得的，可并不是所有人都能耐心等到那个时候。

书中有很多普通人通过努力突破心障从而活出自己想要的样子的例子，有人说这是本鸡汤书，而我更喜欢“自助文学”这个字眼，学习、成长不关他人，只是每个有为的年轻人自己的事情。但行好事，莫问前程。
