02.HTML编辑器——｛HTML入门｝

编写 HTML 的工具：记事本（PC端）、文本编辑器（MAC）

网页可以在专业的 HTML 编辑器进行创建和修改。

然而，在学习 HTML 时，我们推荐用最简单的 文本编辑工具 ，例如 记事本（PC）或 文本编辑器 （MAC）

我们相信在学习 HTML 时用最简单的文本编辑器是一个好的方法。

跟着下面四步来创建你的第一个网页吧！

Step 1: 打开 记事本（PC）

Windows 8 及以后版本：
打开 开始 按钮（ Windows 标志在屏幕左侧的底部地方）。再输入 Notepad 。

Windows 7 及之前版本：
打开 开始 > 程序 > 附件 > 记事本

Step 1: 打开 文本编辑器（MAC）

打开 Finder > 应用 > 文本编辑器

还需修改一些设置确保文件保存为正确格式。在 首选项 > 格式 > 选择 “纯文本” ，在 “ 打开 以及 保存 “ 选项中找到 “在 HTML 文件里忽略富文本命令”。

然后再打开一个新的文档进行编写。

Step 2: 编写一点 HTML

编写 或者 拷贝 一些 HTML 到 记事本里。

Step 3: 保存这个 HTML 页面

保存这个文件到你的电脑里。选择 文件 > 另存为

重命名为 “index.htm” 再选择编码方式为 “UTF-8”

你也可以用 .htm 或 .html 作为文件拓展名。它们没有任何区别，完全随你所选。

Step 4: 在你的浏览器里查看这个 HTML 页面

用你最常用的浏览器打开这个保存过的 HTML 文件（双击这个文件，或者右击 － 选择 “打开方式…”）
